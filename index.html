Hola, mi nombre es Gaston Cazaubon. He trabajado como ingeniero de control de calidad durante los últimos 7 años. Soy un profesional certificado por ISTQB y mi última experiencia fue en TRUX, una empresa con sede en Boston. Durante mis 5 años en TRUX, adquirí experiencia en varias áreas de pruebas: Pruebas manuales: Ejecución de casos de prueba para identificar defectos. Pruebas de API: Uso de Postman para probar API y validar respuestas. Pruebas de bases de datos: Uso de herramientas como DBeaver para consultar y analizar bases de datos. Pruebas de aplicaciones móviles: Pruebas de aplicaciones móviles nativas en diferentes dispositivos y plataformas. Pruebas de aplicaciones web: Pruebas de aplicaciones web en varios navegadores y dispositivos.

    Technical Skills:

  • API Testing & Automation (Postman)
  • Database Testing (SQL Server, Sybase)
  • Mobile Application Testing (iOS, Android)
  • Web Application Testing
  • Tools: JIRA, XRay, Charles Proxy, Bugzilla, Android Studio Emulators, JMeter

    Soft Skills:

  • Strong Collaboration and Communication Skills
  • Analytical Thinking and Problem-Solving
  • Adaptability in Fast-Paced Environments
  • Proactive Team Player with Leadership Abilities
  • Detail-Oriented and Focused on Quality

    Language Skills:

  • Spanish: Native Speaker
  • Experience in international collaborations and client communication
